  • About

PyGoJs is een project van 3 leerlingen van het Alfa-College in Nederland. Het PyGoJs project is voor scholen waarbij de leerlingen bij de les kunnen inchecken met een NFC-Chip bij het begin van de les waarvan het dan geregistreerd zal worden op de server zodat leeraren kunnen zien hoeveel mensen aanwezig waren op de bepaalde les.

Je kan inchecken in een klas door je pas te houden bij de NFC-Scanner die in de klas is geplaatst. De NFC-Scanner is gemaakt met een Raspberry Pi waarbij de gegevens worden verstuurd naar de server. Daar word bekeken of je telaat bent of als je optijd bent.

Frequently Asked Questions

Wat voor kaarten worden er gebruikt dat een NFC-Chip heeft?

De school kaarten van het Alfa-College worden hierbij gebruikt. De studenten krijgen deze pasjes aan het begin van het jaar van de leraren.

Moet bij elke les iedereen aanmelden?

Als je les hebt in dezelfde klas nadat je er nog les had van tevoren, dan hoef jij jezelf niet opnieuw aan te melden. Maar als er een pauze tussen zit, dan moet je wel weer aanmelden met je kaart.

Waar staat "PyGoJs" voor?

PyGoJs staat voor 3 coderings talen waar dit project is mee gemaakt. Python voor de client, GoLang voor de server en Javascript voor de website.

Worden er later ook meer klassen toegevoegd aan het systeem?

Sinds het nu nog maar een klein project is worden er niet meer klassen toegevoegd.
